# ⚔️ JDT 对战游戏

一个基于 Python + WebSocket 的实时双人对战游戏，支持**动态技能扩展**。

玩家通过**聚气（J）、防御（D）、进攻（T）**进行对战，所有操作在浏览器中完成，无需安装任何客户端。

---

## 🎮 游戏规则

### 基础操作

| 按键 | 名称 | 效果           | 消耗   |
| ---- | ---- | -------------- | ------ |
| J    | 聚气 | 获得 1 点 J 气 | 无     |
| D    | 防御 | 抵挡 2 点伤害  | 无     |
| T    | 进攻 | 造成 2 点伤害  | 1 J 气 |

### 对战逻辑

1. 双方**同时出招**，系统自动结算；
2. A方出**攻击招式**，B方出**防御招式**，则判定A方对B方造成`max(0,attack(A)-defense(B))`点伤害；
3. 双方都出**攻击招式**时，数值大者对小者造成`abs(attack(A)-attack(B))`点伤害，双方攻击招式伤害力度相同则**不会造成伤害**；
4. J 气不足时，对应技能按钮自动禁用；
5. 生命值归零时游戏结束。

### 胜负判定

- 一方生命值 ≤ 0 时，另一方获胜。
- 双方可进行多轮对战，直到分出胜负。

## 🚀 快速开始

### 1. 安装 Python

确保你的电脑已安装 **Python 3.8 或更高版本**：

```bash
python --version
```

如果没有安装，去 [python.org](https://www.python.org/downloads/) 下载安装。

### 2. 下载项目

```bash
git clone https://github.com/fzydesu/jdt-game.git
cd jdt-game
```

或者直接下载 ZIP 压缩包解压。

### 3. 安装依赖

```bash
pip install flask flask-socketio eventlet
```

如果下载慢，用国内镜像：

```bash
pip install flask flask-socketio eventlet -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 4. 启动服务

```bash
python app.py
```

看到以下输出表示启动成功：

```
╔═══════════════════════════════════════╗
║   🎮 JDT 对战服务器启动中...          ║
║   访问地址: http://localhost:5000     ║
║   局域网访问: http://你的IP:5000      ║
╚═══════════════════════════════════════╝
```

### 5. 开始游戏

打开浏览器，访问 `http://localhost:5000`

**本地双人对战**：

- 打开两个浏览器窗口
- 两个窗口都点击 **"匹配对手"**
- 匹配成功后，双方选择技能出招

**局域网联机**：

- 查看服务端电脑的 IP 地址（`ipconfig` 或 `ifconfig`）
- 其他设备访问 `http://服务器IP:5000`

## 🛠️ 扩展技能

所有技能定义在 `config.py` 的 `ACTION_TYPES` 字典中。

### 技能配置字段说明

| 字段          | 类型   | 说明                       |
| ------------- | ------ | -------------------------- |
| `name`        | string | 技能名称（显示在按钮上）   |
| `cost`        | int    | 消耗的 J 气数量            |
| `gain`        | int    | 获得的 J 气数量            |
| `damage`      | int    | 造成的伤害值               |
| `defense`     | int    | 防御值（可抵消伤害）       |
| `description` | string | 技能描述（显示在按钮下方） |

### 示例：新增"必杀技"

在 `config.py` 中添加：

```python
ACTION_TYPES = {
    # ... 已有技能 ...
    
    'X': {
        'name': '必杀技',
        'cost': 3,
        'gain': 0,
        'damage': 5,
        'defense': 0,
        'description': '消耗3J，造成5点伤害'
    }
}
```

**重启服务后，前端按钮会自动生成**，无需修改 HTML。

### 图标映射

新技能的图标需要在 `app.py` 的 `actionEmoji` 中添加：

```javascript
const actionEmoji = {
    'J': '🌀',
    'D': '🛡️',
    'T': '⚡',
    'X': '💥',  // 新增
};
```

## 🌐 内网穿透（外网联机）

如果需要让外网朋友访问，推荐使用 **SakuraFrp樱花内网穿透**：

1. 下载SakuraFrp启动器，注册账号，获取访问密钥；
2. 创建新的隧道，选择本地主机名（localhost），本地端口填写`5000`；
3. 设置自动HTTPS为**自动模式**；
4. 复制生成的公网地址（如 `frp-run.com:xxxxx`）发给朋友

## 🐛 常见问题

### Q: 两个浏览器匹配不上？

**A**: 确保两个浏览器都访问同一个地址（`localhost:5000` 或同一个 IP），并且服务端没有报错。

### Q: 其他设备连不上？

**A**: 

1. 检查服务端防火墙是否开放 5000 端口
2. 确保 `app.py` 中 `host='0.0.0.0'`（已默认配置）
3. Windows 用户需在防火墙添加入站规则（TCP 5000）

### Q: 安装依赖报错？

**A**: 尝试使用国内镜像源：

```bash
pip install flask flask-socketio eventlet -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q: 加新技能后按钮没出现？

**A**: 

1. 确认已修改 `config.py` 中的 `ACTION_TYPES`
2. **重启服务**（Ctrl+C 停止，再 `python app.py`）
3. 刷新浏览器页面

### Q: 出招后看不到对手的招式？

**A**: 

1. 确认双方都已出招（状态栏显示"请出招"）
2. 刷新浏览器页面重新匹配
3. 查看浏览器控制台（F12）是否有报错
